# 📊 Stock Market Trend Analysis (Indian Equities)

## 📌 Overview

This project analyzes historical stock price data of major Indian companies to identify trends, compute technical indicators, and forecast short-term price movements using time-series modeling.

It is an end-to-end financial analytics pipeline covering data collection, preprocessing, feature engineering, modeling, and dashboard visualization.

---

## 🎯 Objectives

* Analyze OHLC stock data for multiple companies
* Compute technical indicators (SMA, RSI, Volatility)
* Visualize trends using candlestick charts
* Build a forecasting model (ARIMA)
* Develop an interactive dashboard
